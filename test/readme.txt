Para executar os testes com o selenium basta:
1- Com o navegador Mozilla Firefox, adicionar a extensão Selenium IDE;
2- Ao abrir a IDE do Selenium, selecionar a opção "Open existent project";
3- Selecionar o arquivo "carteira-de-investimentos.side" na pasta "test";
4- Executar os testes, um a um, respeitando a numeração em seus nomes.
